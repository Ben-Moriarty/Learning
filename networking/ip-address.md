Quid est?
An ip address is a logical address for a physical (obstensibly) device. The schema for naming ips is built around the idea of a network (devices in different physical locations connected through some means). An ip is logical because it is not "stuck" to a physical device
like a MAC address. ips can be any value that is 1. Not currently being used 2. within the bounds of usable ips. The addresses themselves are either assigned dynamically (DHCP) or statically.

Why?
ip adresses allow devices to communicate accross different networks. It is essential to divide devices into networks to prevent latency and network congestion.
